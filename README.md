## Прогнозирование выживаемости на Титанике с помощью машинного обучения

## Подключение библиотек

На этом этапе я импортирую необходимые библиотеки, которые играют ключевую роль в улучшении возможностей по обработке и визуализации данных:

* **pandas**: универсальная библиотека для манипуляции и анализа данных, позволяющая эффективно работать с табличными структурами данных.
* **numpy**: фундаментальная библиотека для численных операций и вычислений, предоставляющая эффективные математические функции для работы с массивами.
* **seaborn**: мощная библиотека для визуализации данных, основанная на matplotlib. Она упрощает создание информативной статистической графики.
* **matplotlib.pyplot**: широко используемая библиотека для создания графиков и диаграмм, необходимая для наглядного представления данных и результатов анализа.

## Загрузка данных

Проект начинается с загрузки данных из CSV-файлов с помощью библиотеки pandas. Данные разделены на обучающий (train) и тестовый (test) наборы. Они содержат ключевую информацию о пассажирах "Титаника", которая служит основой для всего анализа.

## Разведочный анализ данных (EDA)

Проводя Разведочный анализ данных (EDA) , я погружаюсь в глубокое изучение набора данных. С помощью различных методов визуализации я:

* **Визуализирую распределения** ключевых числовых признаков, таких как возраст, количество братьев/сестер/супругов (SibSp), количество родителей/детей (Parch) и стоимость билета (Fare). Это помогает выявить закономерности и обнаружить выбросы.
* **Исследую взаимосвязи** между различными переменными, что потенциально позволяет обнаружить корреляции, важные для прогнозирования выживаемости.

## Очистка данных

Очистка данных – ключевой этап анализа. На этой фазе я:

* **Аккуратно обрабатываю пропущенные значения,** выбирая подходящие методы, такие как импутация (заполнение пропусков), чтобы восполнить пробелы в данных.
* Удаляю **неинформативные колонки,** такие как "PassengerId", "Cabin", "Name" и "Ticket", так как они не вносят значимого вклада в анализ.
* Занимаюсь **инженерией признаков (feature engineering)**, создавая новые характеристики или преобразуя существующие для обогащения набора данных. Это может привести к повышению качества предсказаний.

## Тестирование моделей

Проект включает в себя тщательное тестирование моделей для поиска наилучшего подхода к предсказанию выживаемости пассажиров. Я оцениваю несколько алгоритмов машинного обучения, включая: Decision Tree (Дерево решений), LGBM, XGBoost, RandomForest (Случайный лес), ExtraTrees и Logistic Regression (Логистическая регрессия). Кульминацией этой работы стала модель с впечатляющей точностью 73.8% , что демонстрирует ее способность делать достоверные прогнозы.

## Формирование файла с результатами (Submission)

Применив отобранную модель к тестовому набору данных, я создаю файл с предсказаниями (submission file). Это критически важный шаг для оценки модели, так как он позволяет увидеть, насколько хорошо она обобщает знания на новых, невиданных ранее данных.

## Заключение

Достигнутая точность прогноза в 73.8% показывает, что модель достаточно эффективно предсказывает выживаемость пассажиров "Титаника". Этот проект не только демонстрирует практическое применение техник машинного обучения на исторических данных, но и дает представление о факторах, которые сыграли ключевую роль в определении шансов на выживание во время катастрофы.
